# Espace Agents — Peya Pay

Cette page regroupe les informations utiles pour les **agents / points de vente** (parcours, bonnes pratiques, et endpoints à utiliser dans le sandbox).

## Accès

- **Sandbox**: utilisez la page `/login` pour obtenir un token (si nécessaire).
- **Navigation**: l’onglet **Agents** est accessible via `/agent`.

## Parcours recommandé

1. **Récupérer un token** (Admin / Partenaire)
2. **Tester les opérations** depuis le sandbox (OTP, recherche client, paiement, etc.)
3. **Consulter la documentation API** si un champ/format n’est pas clair

## Liens utiles

- Documentation API: `/docs`
- Sandbox: `/login`
